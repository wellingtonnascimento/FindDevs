## **Back-end**

Desenvolvido utilizado node.js para criar uma API REST

### **End Points**

- **Cadastrar usu치rio**

  **URL:** `/devs`

  **Method:** `POST`

- **Listar todos os usu치rios**

  **URL:** `/devs`

  **Method:** `GET`

- **Atualizar usu치rio pelo ID**

  **URL:** `/devs/:id`

  **Method:** `PUT`

  **Required:** `id=[integer]`

- **Apagar usu치rio pelo ID**

  **URL:** `/devs/:id`

  **Method:** `DELETE`

  **Required:** `id=[integer]`
